📘 Series Sum Programs in C

This repository contains simple C programs to calculate different mathematical series:

- 🔢 Sine Series (sin x)
- 🔢 Cosine Series (cos x)
- 🔢 Sum of Series (general)

These programs help in understanding loops, functions, and Taylor series expansion in C.

---

📂 Contents

Program| Description
"sin_series.c"| Calculates sin(x) using Taylor series
"cos_series.c"| Calculates cos(x) using Taylor series
"sum_series.c"| Calculates sum of a given series

---

🧮 Formulas Used

🔹 Sine Series

sin(x) = x - x³/3! + x⁵/5! - ...

🔹 Cosine Series

cos(x) = 1 - x²/2! + x⁴/4! - ...

---

⚙️ How to Run

1. Clone the repository:

git clone https://github.com/your-username/your-repo-name.git

2. Navigate to the folder:

cd your-repo-name

3. Compile the programs:

gcc sin_series.c -o sin
gcc cos_series.c -o cos
gcc sum_series.c -o sum

4. Run the programs:

./sin
./cos
./sum
